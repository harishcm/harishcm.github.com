Prior Art

Daring Fireball

The layout and dark background of this website was partially inspired by Daring Fireball. I also use Markdown, which was created by the author, when writing for this website.

Desert Colourscheme for Vim

The colourscheme for this website is based on the Desert colourscheme for the Vim text editor. I found it be very pleasant to the eye and to facilitate reading.

Tools

Markdown

Several of the webpages in this site are authored in Markdown. Markdown is a plain text syntax for authoring webpages. It is also a tool for converting the syntax into html pages. Markdown aims to be extremely readable in its source form. The syntax resembles plain text emails and bulletin board messages.

VimOutliner

VimOutliner is an outline editor addon to the Vim text editor. It is lightning fast in outline creation, and section collapsing and expanding. It also differentiates between headings and text. I use VimOutliner for almost all of my digital record keeping.

Ikiwiki

The website engine generating the html pages from the Markdown and VimOutliner sources.

Firebug

Firebug is an addon for the Firefox web browser. It integrates several web development tools into the web browser itself. I use it extensively when modifying my CSS and template files.

Open Library Covers Repository

Book cover images in this website are downloaded through the Open Library Covers API. Open Library is a non-profit open-data project attempting to build a freely accessible online book catalogue of all books ever published.

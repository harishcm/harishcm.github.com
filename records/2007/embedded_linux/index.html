Embedded Linux

  1. Inspiration

      I had attended a talk on Embedded Linux. I was impresssed by the speaker’s enthusiasm in putting Linux on as many computer boards (of different architectures) as he could. To me, it was an interesting extension of my day to day Linux usage. Moreover, embedded device usage was growing at a fast rate in several fields.

      I want to know how one creates embedded devices and building my own Linux embedded system is a very possible way of doing so.

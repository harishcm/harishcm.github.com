    2. Exploration
      1. Unix Scheduling Programs
        1. cron
        2. anacron
        3. at
      2. Typical differences between cron, anacron and at

          cron runs sheduled tasks at the specific times up to intervals of one minute apart. If the computer was not up, the sheduled tasks will not run. More approporiate for

          anacron runs sheduled tasks at the closest time possible if the machine was not up. Tasks can be specified only to hourly intervals. More appropraite for desktop users or servers that are not to be up all the time.

          at is used to shedule a single task at a specific time. It can also accept a text file containing sheduled tasks as input. It is invoked by the atd daemon.

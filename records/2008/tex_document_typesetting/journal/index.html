  3. 2010-09-07
    1. Added Font Packages to Outline
      1. cmbright

          Computer Modern Bright font. It is a sans font with a corresponding math font that was derived from Computer Modern.

      2. mathptmx

          Times font with a corresponding math font.

      3. iwona or kurier

          Sans font with a corresponding math font. Needs to be specified with [math] option for corresponding math font to be enabled. The only difference is that kurier has gaps in its strokes.

      4. anttor

          Stylish serif font with a corresponding math font. Needs to be specified with [math] option for corresponding math font to be enabled.

      5. fouriernc

          Uses New Century Schoolbook as the text font and for Latin letters in math. Uses Greek letters and symbols from the Fourier-GUTenberg package.

      6. fourier

          Similar to fouriernc but uses Utopia as the text font and for Latin letters in Math.

      7. arev

          Sans font with math support. Uses an extended version of Bitstream Vera Sans called Arev Sans for Greek and many mathematical symbols. Uses bold version of Bitstream Charter for symbols.

  16. 2008-09-10
    1. Achievement
      1. Typeset a lab report within 1.5 days

          Experience and example tex sources from the prior report done during the UROP project allowed me to have a base to start off from.

          Challenges were typesetting the numerous equations and fitting a (horizontally) wide table into a page. I eventually put that set that page in landscape using the “lscape” package and increased the borders by using the “fullpage” package.

        1. Evaluation

            The charts and most of the general typesetting was good. There was a page of short equations that seemed to have extremely large left and right margins. Placing a whole page of short equations in two column format may make it better.

        2. Evaluation
          1. 2008-09-22

              I still do not have much experience with more advanced math typesetting. There are still a lot more commands (though simple and self-evident in many cases) to be learnt.

TeX Document Typesetting

  1. Inspiration

      I heard about the beauty of typesetting done by TeX and LateX during a social interaction session after a Singapore Linux Meetup Group (SLMG) meetup. This was perfectly in tune with my prior interest in typography and in creating a Tamil font.

      I also heard that learning LyX would be a good way to ease the transition from document prepartion with word processors to document preparationwith LateX. Another draw was the possibility for extended separation of content from style.

  2. Packages I have used
    1. Tables
      1. booktabs

          For creating standard publication quality tables.

      2. tabularx

          For creating fixed width tables with columns of variable size. Usual fixed width tables in latex vary the spacing between columns instead.

      3. longtable

          Allows for tables spanning multiple pages.

      4. ltxtable

          Allows the usage of both tabularx and longtable together.

      5. dcolumn

          To easily align numbers along the decimal point.

      6. multirow

          To create cells that span multiple rows. The \minitab command used in the package readme allows linebreaks in multirow cells. Use with a negative nrows for cells to align with bottom row. For negative nrows, data must be entered in the bottom row.

    2. Floats
      1. wrapfig

          Enables wrapping of text around figures, tables and other floats.

      2. caption

          Allows for changing of several style parameters for the captions.

      3. float

          Provides a ’H’ option to force placement of floats exactly at the position that they appear in the document file.

    3. Layout
      1. geometry

          Full control of layout options for whole document

      2. lscape

          Provides an environment to place parts of the document in landscape mode.

      3. setspace

          Makes changing of linespacing easier.

    4. Fonts
      1. bera

          Bitstream Vera fonts. Suitable for screen reading. Redefines serif, sans-serif and typewriter fonts.

      2. charter

          Bitstream Charter fonts. Slabs serifs make it suitable for screen reading. Redefines only serif fonts.

      3. mathdesign

          Math fonts that complement Bitstream Charter [charter], Adobe Utopia [utopia] and URW Garamond [garamond] text fonts.

      4. cmbright

          Computer Modern Bright font. It is a sans font with a corresponding math font that was derived from Computer Modern.

      5. mathptmx

          Times font with a corresponding math font.

      6. iwona or kurier

          Sans font with a corresponding math font. Needs to be specified with [math] option for corresponding math font to be enabled. The only difference is that kurier has gaps in its strokes.

      7. anttor

          Stylish serif font with a corresponding math font. Needs to be specified with [math] option for corresponding math font to be enabled.

      8. fouriernc

          Uses New Century Schoolbook as the text font and for Latin letters in math. Uses Greek letters and symbols from the Fourier-GUTenberg package.

      9. fourier

          Similar to fouriernc but uses Utopia as the text font and for Latin letters in Math.

      10. arev

          Sans font with math support. Uses an extended version of Bitstream Vera Sans called Arev Sans for Greek and many mathematical symbols. Uses bold version of Bitstream Charter for symbols.

    5. Hyphenation
      1. hyphenat

          Options to disable hyphenation among others.

    6. Lists
      1. enumitem

          Change various parameters of enumerate, itemize and description lists.

    7. Line breaking
      1. url

          Breaks urls cleanly. The “hyphens” option breaks urls at hyphens as well. Use with \raggedright if urls still overflow to the right.

  3. Useful commands
    1. Tables
      1. \newcolumntype{C}{>{\centering\arraybackslash}X}

          For centered tabularx columns

      2. \minitab[c]{Some \ linebroken \ text}

          Used with the multirow package to enable line breaking in cells spanning multiple rows.

        1. \newcommand{\minitab}[2][l]{\begin{tabular}{#1}#2\end{tabular}}
    2. Layout
      1. \enlargethispage{N\baselineskip}

          Used to extend the page by the length specified as the argument. Change N to be the number of lines more to be included in the current page.

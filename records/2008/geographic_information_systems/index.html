    4. Geocoding
      1. Google Maps
        1. Request url
          1. http://maps.google.com/maps/geo?q=ENCODED_STRING&output=xml

              Change the ENCODED_STRING to the address to be geocoded. Address can be typed directly in a web browser. Fairly accurate for Singapore addresses but only so-so for Singapore postal codes.
